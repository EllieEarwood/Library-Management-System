Library Management System
InsertBook.java

Description:
This script allows the user to manually input book details and adds a new record to the `books` table in the database.

Steps to Use:

1. Make sure your MySQL server is running and the `librarydb` database is set up.
2. Ensure the `DatabaseConnection` class is included in the project and configured with your correct database credentials.
3. Open NetBeans and add InsertBook.java to your project.
4. Run the file.
5. Enter the book title, author, and availability status when prompted in the console.

The script will insert the book into the database and confirm whether the operation succeeded.
